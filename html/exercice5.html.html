Les liste

Les listes permettent de structurer les informations sous forme de liste. Il existe deux types de listes: les listes ordonnées et les listes non-ordonnées.

liste non-ordonnees

J'ai envie d'apprendre les langages suivants:

    HTML
    CSS
    PHP
    JAVASCRIPT

les liste ordonnees

J'ai envie d'apprendre les langages suivants:

    1.HTML

    2.CSS

    3.PHP

    4.JAVASCRIPT
